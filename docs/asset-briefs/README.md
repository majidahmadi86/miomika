# Asset briefs

Each file in this directory is a complete prompt for ChatGPT / DALL-E /
Midjourney to generate a specific Miomika asset.

The canonical asset list is in `/MIOMIKA.md` §7. Each asset brief here
corresponds to one asset in that list.

## Format

Every brief contains:
- Asset name and exact dimensions
- Composition rules
- Color palette (locked to design tokens)
- "Avoid" list (no frames, no shadows, transparent bg)
- Reference to existing Miomi style
- Output format and file path

## How to use

1. Open ChatGPT (or your image generator of choice).
2. Paste the brief contents.
3. Generate and refine the image.
4. Download as PNG with transparent background.
5. Save to the path specified in the brief.
6. Commit.

Asset briefs are generated by the phase prompts that need them.
For example, Phase 1's prompt generates `favicon-brief.md`,
`companion-idle-brief.md`, `companion-happy-brief.md` etc.
