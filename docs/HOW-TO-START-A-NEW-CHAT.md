# How to start a new Claude.ai chat for Miomika

When starting a fresh Claude.ai chat to continue Miomika development:

## Step 1 — Paste this opening message

```
Hi Claude. I'm continuing work on Miomika, my AI companion + language learning product.

Before we do anything else, please:
1. Read /MASTER-HANDOFF.md at my project root — it has founder context, project story, my communication style, and current state.
2. Read /MIOMIKA.md at my project root — it's the engineering contract with the full execution plan, design tokens, and Codebase Map (§11).
3. Once you've read both, confirm with: "I've read both documents. Current state: end of Phase [X]. Ready to proceed with Phase [Y]?"

Don't ask me background questions. Everything is in those two files.
Don't propose changes to the project story or design rules — those are locked.
Just confirm context, then ask what we're working on this session.
```

## Step 2 — Attach the two files

Drag `/MASTER-HANDOFF.md` AND `/MIOMIKA.md` into the chat. Claude will then have the full context.

## Step 3 — Tell Claude what we're doing

Examples:
- "Generate the Phase 3B master-prompt"
- "Fix bug: [paste screenshot]"
- "Add a new feature: [describe]"

Claude will write a Cursor master-prompt. Run it. Come back with results.

## Don't ever paste this protocol — it's the protocol, not the prompt
