# Architecture Document Archive

Versioned snapshots of major architecture documents. **Never deleted.** Any future major rewrite saves the previous version here first.

## Why this exists

Mike has lost context across chat sessions when documents got rewritten without archiving the previous version. From v6 forward, every major MIOMIKA.md or SCREENS.md rewrite saves a timestamped copy here BEFORE editing.

## Policy

- Save before edit. Always.
- Format: `<DOC>-v<N>-<YYYY-MM-DD>.md`
- Add a row to this index table.
- Never modify or delete archived files. They are read-only history.
- Use `git log` for fine-grained edits between major versions.
